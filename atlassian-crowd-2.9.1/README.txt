Atlassian Crowd 2.9.1
----------------------------------------------

Thank you for downloading Atlassian Crowd 2.9.1.

QUICK START GUIDE
-----------------

1. Ensure you have a JDK installed.

2. Set the JAVA_HOME variable to where you installed Java.

2. Set your Crowd home directory.
   Edit the file 'crowd-webapp/WEB-INF/classes/crowd-init.properties'
   and set crowd.home to a writeable directory.

3. Run 'start_crowd.sh' (or 'start_crowd.bat' for Windows).

4. Point your browser to http://localhost:8095/crowd/ to see Crowd's
   setup wizard.

Full documentation is available online at:

    https://confluence.atlassian.com/display/CROWD/Installing+Crowd
